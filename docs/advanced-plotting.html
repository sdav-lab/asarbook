  • When histograms look too choppy
  • Showing overlapping distributions

9.4 Text Labels with geom_text_repel()

geom_text_repel() intelligently positions text labels to avoid overlaps.

9.4.1 Why Use geom_text_repel?

Standard geom_text() often creates overlapping labels. geom_text_repel() from the ggrepel package automatically adjusts positions.

9.4.2 Basic Text Labels

# Regular geom_text (overlapping)
ggplot(players, aes(x = points_per_game, y = salary_millions)) +
  geom_point() +
  geom_text(aes(label = player), size = 2, hjust = 1) +
  labs(title = "Without Repel: Overlapping Labels") +
  theme_minimal()

9.4.3 Repelled Text Labels

# geom_text_repel (intelligent positioning)
ggplot(players, aes(x = points_per_game, y = salary_millions)) +
  geom_point(size = 2, alpha = 0.6) +
  geom_text_repel(aes(label = player), size = 2.5, max.overlaps = 20) +
  labs(title = "With Repel: Clean, Readable Labels",
       x = "Points per Game",
       y = "Salary ($ millions)") +
  theme_minimal()
