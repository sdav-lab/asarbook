4.1 Chapter Overview

Topics covered:

  • Measures of central tendency (mean, median, mode)
  • Measures of dispersion (range, variance, standard deviation)
  • Frequency tables and cross-tabulations
  • Percentiles and quartiles
  • Creating summary tables

By the end of this chapter, you’ll be able to:

  1. Calculate and interpret measures of central tendency
  2. Assess data spread using variance and standard deviation
  3. Create frequency tables for categorical data
  4. Generate cross-tabulations
  5. Calculate percentiles and quartiles
  6. Use AI to help interpret statistical findings

4.2 Measures of Central Tendency

Central tendency describes the “typical” or “center” value of a dataset.

4.2.1 Mean (Average)

The mean is the sum divided by the count:

# Game attendance
attendance <- c(1234, 1567, 1432, 1890, 1654, 1234, 1765, 1543)

# Calculate mean
mean(attendance)
## [1] 1539.875
# With missing values
attendance_na <- c(1234, NA, 1432, 1890, 1654)
mean(attendance_na, na.rm = TRUE)
## [1] 1552.5

When to use: Numeric data without extreme outliers

4.2.2 Median (Middle Value)

The median is the middle value when sorted:

# Calculate median
median(attendance)
## [1] 1555
# Median is resistant to outliers
data_with_outlier <- c(10, 12, 15, 18, 20, 1000)
mean(data_with_outlier)    # Pulled up by outlier
## [1] 179.1667
median(data_with_outlier)  # Not affected
## [1] 16.5

When to use: When data has outliers or is skewed

4.2.3 Mode (Most Frequent)

The mode is the most common value. R doesn’t have a built-in mode function:

# Find mode manually
positions <- c("G", "F", "C", "G", "F", "G", "C", "G")

# Create frequency table
table(positions)
## positions
## C F G 
## 2 2 4
# G appears most (4 times), so mode is "G"

# Function to find mode
get_mode <- function(x) {
  unique_x <- unique(x)
  unique_x[which.max(tabulate(match(x, unique_x)))]
}

get_mode(positions)
## [1] "G"

When to use: Categorical data

4.2.4 Comparing Central Tendency Measures

# Symmetric data - all three similar
symmetric <- c(10, 12, 14, 15, 16, 18, 20)
mean(symmetric)
## [1] 15
median(symmetric)
## [1] 15
# Right-skewed data - mean > median
right_skewed <- c(10, 12, 14, 15, 16, 18, 100)
mean(right_skewed)
## [1] 26.42857
median(right_skewed)
## [1] 15
# Left-skewed data - mean < median
left_skewed <- c(1, 82, 84, 85, 86, 88, 90)
mean(left_skewed)
## [1] 73.71429
median(left_skewed)
## [1] 85

4.3 Measures of Dispersion

Dispersion describes how spread out the data is.

4.3.1 Range

The range is the difference between maximum and minimum:

points <- c(78, 82, 71, 88, 76, 85, 79)

# Get min and max
min(points)
## [1] 71
max(points)
## [1] 88
# Calculate range
range_value <- max(points) - min(points)
range_value
## [1] 17
# range() returns both min and max
range(points)
## [1] 71 88

4.3.2 Variance

Variance measures average squared deviation from the mean:

# Calculate variance
var(points)
## [1] 32.47619
# What it means:
# 1. Find mean
# 2. Find difference from mean for each value
# 3. Square those differences
# 4. Average the squared differences

Problem: Variance is in squared units (hard to interpret)

4.3.3 Standard Deviation

Standard deviation is the square root of variance:

# Calculate standard deviation
sd(points)
## [1] 5.698789
# Relationship to variance
sqrt(var(points))
## [1] 5.698789
# Interpretation: On average, values are about 5.4 points
# away from the mean

Rule of thumb: About 68% of data falls within 1 SD of the mean (for normal distributions)

4.3.4 Coefficient of Variation

CV expresses SD as a percentage of the mean:

# Calculate CV
points_cv <- (sd(points) / mean(points)) * 100
points_cv
## [1] 7.136229
# Useful for comparing variability across different scales

4.4 Frequency Tables

Frequency tables show counts for categorical data.

4.4.1 Simple Frequency Tables

# Player positions
positions <- c("G", "F", "C", "G", "F", "G", "C", "G", "F", "F")

# Create frequency table
freq_table <- table(positions)
freq_table
## positions
## C F G 
## 2 4 4
# Convert to percentages
prop.table(freq_table) * 100
## positions
##  C  F  G 
## 20 40 40
# Round percentages
round(prop.table(freq_table) * 100, 1)
## positions
##  C  F  G 
## 20 40 40

4.4.2 Using dplyr

# Create data frame
players <- data.frame(
  name = paste("Player", 1:10),
  position = c("G", "F", "C", "G", "F", "G", "C", "G", "F", "F")
)

# Count with dplyr (use explicit namespace to avoid conflicts)
players %>%
  dplyr::count(position) %>%
  mutate(percentage = n / sum(n) * 100)
##   position n percentage
## 1        C 2         20
## 2        F 4         40
## 3        G 4         40

4.4.3 Cross-Tabulations

Cross-tabs show relationships between two categorical variables:

# Create data
team_data <- data.frame(
  division = c("I", "I", "II", "II", "III", "III", "I", "II"),
  classification = c("Public", "Private", "Public", "Private",
                     "Public", "Private", "Public", "Private")

# Create cross-tabulation
table(team_data$division, team_data$classification)
##      
##       Private Public
##   I         1      2

##   I         1      2   3
##   II        2      1   3
##   III       1      1   2

4.5 Percentiles and Quartiles

Percentiles divide data into 100 equal parts.

4.5.1 Calculating Percentiles

scores <- c(65, 72, 78, 81, 84, 87, 89, 91, 93, 95)

# 50th percentile (median)
quantile(scores, 0.50)
##  50% 
## 85.5
# 25th and 75th percentiles
quantile(scores, c(0.25, 0.75))
##   25%   75% 
## 78.75 90.50
# Quartiles (0%, 25%, 50%, 75%, 100%)
quantile(scores)
##    0%   25%   50%   75%  100% 
## 65.00 78.75 85.50 90.50 95.00
# Specific percentiles
quantile(scores, c(0.10, 0.90))  # 10th and 90th
##  10%  90% 
## 71.3 93.2

4.5.2 Interquartile Range (IQR)

IQR is the range of the middle 50% of data:

# Calculate IQR
IQR(scores)
## [1] 11.75
# Manually
q1 <- quantile(scores, 0.25)
q3 <- quantile(scores, 0.75)
q3 - q1
##   75% 
## 11.75

4.5.3 Five-Number Summary

# Min, Q1, Median, Q3, Max
fivenum(scores)
## [1] 65.0 78.0 85.5 91.0 95.0
# Or use summary()
summary(scores)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##   65.00   78.75   85.50   83.50   90.50   95.00

4.5.4 Identifying Outliers

Rule: Values beyond Q1 - 1.5×IQR or Q3 + 1.5×IQR are potential outliers

# Calculate outlier boundaries
q1 <- quantile(scores, 0.25)
q3 <- quantile(scores, 0.75)
iqr <- IQR(scores)

lower_bound <- q1 - 1.5 * iqr
upper_bound <- q3 + 1.5 * iqr

# Find outliers
scores[scores < lower_bound | scores > upper_bound]
## numeric(0)

# Load season data
season_stats <- data.frame(
  game = 1:20,
  points_for = c(78, 82, 71, 88, 76, 85, 79, 91, 74, 83,
                 77, 86, 81, 89, 75, 80, 84, 73, 87, 82),

                     75, 83, 79, 85, 78, 77, 81, 75, 84, 79)
)

# Calculate statistics
cat("=== Offensive Statistics ===\n")
## === Offensive Statistics ===

## Mean PPG: 81.05

## Median PPG: 81.5

## SD: 5.661272

## Range: 71 - 91
cat("IQR:", IQR(season_stats$points_for), "\n\n")
## IQR: 8.5
cat("=== Defensive Statistics ===\n")
## === Defensive Statistics ===

## Mean Points Allowed: 79.15

## Median Points Allowed: 79

## SD: 4.145702

## === Point Differential ===
diff <- season_stats$points_for - season_stats$points_against

## Mean Margin: 1.9

## Median Margin: 3
cat("SD:", sd(diff), "\n")
## SD: 2.954034
# Consistency analysis

## === Consistency ===
cat("Offensive CV:", (sd(season_stats$points_for) / mean(season_stats$points_for)) * 100, "%\n")
