8.3 Histograms: Visualizing Distributions

Histograms show the distribution of a continuous variable by dividing data into bins and counting observations in each bin.

8.3.1 When to Use Histograms

  • Exploring the shape of a distribution (normal, skewed, bimodal)
  • Identifying outliers
  • Understanding data spread and central tendency
  • One continuous variable

8.3.2 Basic Histogram

# Distribution of points scored
ggplot(unf_season, aes(x = points_scored)) +
  geom_histogram(fill = "steelblue", color = "white") +
  labs(title = "Distribution of Points Scored",
       x = "Points Scored",
       y = "Frequency") +
  theme_minimal()

8.3.3 Controlling Bin Width

# Specify number of bins
ggplot(unf_season, aes(x = points_scored)) +
  geom_histogram(bins = 10, fill = "steelblue", color = "white") +
  labs(title = "Points Scored (10 bins)") +
  theme_minimal()

# Specify bin width
ggplot(unf_season, aes(x = points_scored)) +
  geom_histogram(binwidth = 5, fill = "steelblue", color = "white") +
  labs(title = "Points Scored (binwidth = 5)") +
  theme_minimal()

Choosing bins: Too few bins hide details, too many create noise. Start with the default, then experiment. A good rule of thumb: √n bins where n = number of observations.

8.3.4 Overlaying Multiple Distributions

# Compare home vs. away scoring
ggplot(unf_season, aes(x = points_scored, fill = location)) +
  geom_histogram(alpha = 0.6, position = "identity", binwidth = 5) +

  labs(title = "Scoring Distribution: Home vs. Away",
       x = "Points Scored",

       fill = "Location") +
  theme_minimal()

  • Ranking items

8.5 Boxplots: Comparing Distributions

Boxplots show the distribution of data through quartiles, highlighting median, spread, and outliers.

8.5.1 When to Use Boxplots

  • Comparing distributions across groups
  • Identifying outliers
  • Showing spread and skewness
  • Multiple groups, one continuous variable

8.5.2 Anatomy of a Boxplot

# Basic boxplot
ggplot(unf_season, aes(x = location, y = points_scored, fill = location)) +
  geom_boxplot() +
  scale_fill_manual(values = c("Home" = "#003366", "Away" = "#CC0000")) +
  labs(title = "Points Scored: Home vs. Away",
       subtitle = "Box shows Q1, median, Q3; whiskers extend to 1.5*IQR",
       x = "Location",
       y = "Points Scored") +
  theme_minimal() +
  theme(legend.position = "none")

Reading a boxplot: - Box edges = 25th and 75th percentiles (Q1 and Q3) - Line inside box = median (50th percentile) - Whiskers = extend to 1.5 × IQR (interquartile range) - Points beyond whiskers = potential outliers

8.5.3 Multiple Group Boxplots

# Boxplot by quarter
ggplot(unf_season, aes(x = quarter, y = points_scored, fill = quarter)) +

  theme(legend.position = "none")

# Boxplot with jittered points
ggplot(unf_season, aes(x = location, y = points_scored, fill = location)) +

  geom_jitter(width = 0.2, alpha = 0.5, size = 2) +

ggplot(unf_season, aes(x = result, y = points_scored, fill = result)) +

Violin plots combine boxplots with density plots, showing the full distribution shape.

8.6.1 When to Use Violin Plots

  • When you want more detail than a boxplot
  • Comparing distributions with different shapes
  • Showing bimodal or multimodal distributions
  • Audiences familiar with advanced visualizations

8.6.2 Basic Violin Plot

ggplot(unf_season, aes(x = location, y = points_scored, fill = location)) +

       x = "Location",
       y = "Points Scored") +

  theme(legend.position = "none")

# Violin + boxplot combo (best of both worlds)
ggplot(unf_season, aes(x = quarter, y = points_scored, fill = quarter)) +

8.9 Choosing the Right Plot Type

8.9.1 Decision Flow

Question 1: What type of variable(s) do you have?

  • One continuous variable → Histogram or density plot
  • One categorical + one continuous → Bar chart or boxplot
  • Two continuous variables → Scatterplot (Chapter 7)
  • Time series → Line plot or area chart
  • Multiple groups + continuous → Boxplot or violin plot

Question 2: What are you trying to show?

  • Distribution shape → Histogram, density plot, violin plot
  • Compare groups → Bar chart, boxplot, violin plot
  • Trend over time → Line plot, area chart
  • Identify outliers → Boxplot
  • Show individual data points → Jittered points with boxplot

8.9.2 Comparison Table

Plot Type Best For Pros Cons
Histogram Single distribution Shows shape clearly Bin choice affects interpretation
Bar Chart Category comparisons Easy to read Only shows summary stats
Boxplot Multiple group comparisons Shows outliers, quartiles Hides distribution shape
Violin Plot Detailed group comparisons Shows full distribution Harder to interpret
Line Plot Time series Shows trends clearly Can be cluttered with many lines
Area Chart Cumulative trends Emphasizes magnitude Harder to compare exact values

8.13 Chapter Summary

You now have a solid toolkit of basic visualizations for sport analytics!

Key Takeaways:

  1. Histograms show distributions of continuous variables
  2. Bar charts compare categories or show counts
  3. Boxplots compare distributions across groups and identify outliers
  4. Violin plots show full distribution shapes (richer than boxplots)
  5. Line plots track trends over time or sequences
  6. Area charts emphasize cumulative trends
  7. Choose plots based on your data type and your question
  8. Combine geoms for richer, more informative visualizations
  9. AI can help with plot selection and code, but verify output

Essential Functions:

Function Purpose
geom_histogram() Distribution of continuous variable
geom_col() / geom_bar() Bar charts for categories
geom_boxplot() Compare distributions, show quartiles
geom_violin() Distribution shapes for comparison
geom_line() Trends over time/sequence
geom_area() Cumulative trends with area fill
geom_jitter() Add scattered points with noise
stat_summary() Add summary statistics to plots
